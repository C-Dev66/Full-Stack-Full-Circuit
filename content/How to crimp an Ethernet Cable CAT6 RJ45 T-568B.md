# How to crimp an Ethernet Cable CAT6 RJ45 T-568B

Today, we successfully crimped our very first CAT6 Ethernet cable (RJ45) using the T-568B pinout standard. Using a known defective roll of Ethernet cable, I was able to salvage and cut five 5-foot strands for practice. Each end was terminated with a CAT6 RJ45 Pass-Through connector using the Klein Tools VDV226-011 crimping tool.

The recommended configuration for most applications is the T-568B straight-through wiring method, which uses the same color code on both ends of the cable. Alternatively, a crossover configuration can be created by wiring one end as T-568B and the other as T-568A.

T-568B Pinout (with locking tab facing down):

| Pin 1        | Pin 2  | Pin 3       | Pin 4 | Pin 5      | Pin 6 | Pin 7       | Pin 8 |
| ------------ | ------ | ----------- | ----- | ---------- | ----- | ----------- | ----- |
| White Orange | Orange | White Green | Blue  | White Blue | Green | White Brown | Brown |
