# How to use Onshape

Onshape default Mouse Controls:

|                    | 3D Part Assembly               | 2D Drawing                     |
| ------------------ | ------------------------------ | ------------------------------ |
| Rotate             | Right Mouse button drag        | N/A                            |
| Constrained rotate | Alt + Right mouse button drag  | N/A                            |
| Pan                | Crtl + Right mouse button drag | Ctrl + Right mouse button drag |
| Zoom               | Scroll wheel in/out            | Scroll wheel in/out            |

In Onshape, **documents** are containers for parts, assemblies, drawings, and other related data. Each one lives within its own **tab** in the document. When creating a new drawing planes will divide the structure: Top, Front, Right. Creating a sketch is the basic component to begin creating a feature. Sketch entities include Lines and Rectangles, Circles and Arcs, Construction of Geometry, Slot, Polygons, Ellipse and Sketch Fillet, Sketch Points. 

When you are finished using a tool press ESC

Construction Geometry: is sketch geometry that is there to help define a sketch but is ignored when creating a feature. Any sketch entities can become Construction geometry.

To round a corner use the sketch fillet tool.
