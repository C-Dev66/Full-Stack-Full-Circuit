# Obsidian The King of Learning Tools (Full Guide + Setup)

As you learn about new topics creating a central repository of notes is a powerful tool that allows you to organize your thoughts and make connections. By allowing your self to slow down an think about what you're writing, new ideas will be discovered. Tags and References will prove useful when linking thoughts and past experiences together. Think of it as a personal writing/idea assistant.

Don't forget, simplicity is key. Keep the note under 500 words. Tackle one idea at a time. 

| Folders             | Purpose                                                                                                                                                                                                                                                                                        |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 01. Rough Notes     | Notepad where you write quick ideas down or To Do's. Return to this area and clean up any open tasks. Goal is to make every note into a #adult with tags and source material.                                                                                                                  |
| 02. Source Material | Area to store and publish thoughts about more formal methods of content. There are separate from the final notes. Create a Main Note from the source material and allow your self to see the bigger picture and ask so what. What did I learn? What's the purpose?                             |
| 03. Tags            | Tags are empty notes that are used later to create an index. Don't be to vague and remember to tailor to your use case and persona. To make a tag encapsulate the phrase in "[ [ ] ]" two brackets. Initially the tag will be grey, If so create into a new note and move to the tags folder.  |
| 04. Indexes         | Once the knowledge base grows and a tag has many connections. You can turn the tag into an index and give it structure. Create subheadings and organize each of the notes.                                                                                                                     |
| 05. Template        | Stores the template used for all your notes                                                                                                                                                                                                                                                    |
| 06. Main Notes      | All notes that aren't source material go here                                                                                                                                                                                                                                                  |

| Steps       | Description                                                                                                              |
| ----------- | ------------------------------------------------------------------------------------------------------------------------ |
| Ctrl+O      | Search bar pops up to look if you have a similar note created. Avoid writing the same note twice.                        |
| Shift+Enter | To create a new note from title typed in the search bar                                                                  |
| Ctrl+t      | To add the default template onto the note                                                                                |
| References  | Create hyper links to any notes with similar ideas. Using the same method as tags "[ [ ]]". Link to the source material  |
